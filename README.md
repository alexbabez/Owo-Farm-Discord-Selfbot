## 🎮 Commands

- **Start the bot:** `.start`
  - This command starts the bot and begins farming OwO commands automatically.
  
- **Stop the bot:** `.stop`
  - This command stops the bot from farming.
  
- **Help:** `.help`
  - Displays a list of available commands and their descriptions.
